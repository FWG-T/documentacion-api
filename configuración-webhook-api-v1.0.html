Webhook Gestión de Notificaciones Contratistas, Vehículos y Personal (1.0.1)

Download OpenAPI specification:Download

Webhook para la entrega y notificación de eventos relacionados con la información de contratistas, personal y vehículos, incluyendo actualizaciones de estado, colores, y otros datos relevantes para el seguimiento y la gestión.

EntityEventWebhook

Webhook para la entrega y notificación de eventos relacionados con la información de contratistas, personal y vehículos, incluyendo actualizaciones de estado, colores, y otros datos relevantes para el seguimiento y la gestión.
